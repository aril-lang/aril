# Task: absolute value, with a contract

Write a function that returns the absolute value of an integer. In addition to
implementing it, **attach a contract** to the function stating the property that
its result is always greater than or equal to zero (this is the kind of
machine-checkable guarantee the language lets you write alongside a function).

Then print the absolute value of -5, of 3, and of 0 — each on its own line, in
that order.

Print nothing else.

## Exact expected output

```
5
3
0
```
