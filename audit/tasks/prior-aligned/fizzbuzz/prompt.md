# Task: FizzBuzz, 1 through 15

Print the integers from 1 to 15 inclusive, one per line, in order — with these
substitutions:

- a number divisible by 3 (but not 5) → print `Fizz`
- a number divisible by 5 (but not 3) → print `Buzz`
- a number divisible by both 3 and 5 → print `FizzBuzz`
- any other number → print the number itself

Print nothing else — no header, no trailing summary.

## Exact expected output

```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
```
