# Task: word counts, including a missing word

Given the text `a b a c b a` (single-space separated), count how many times each
word appears. Then print, each on its own line and in this order, the count of:
`a`, then `b`, then `c`, then `z`.

Note that `z` never appears in the text — its count is 0, and looking it up must
not crash the program.

Print nothing else.

## Exact expected output

```
3
2
1
0
```
