# Task: collect the squares

Build a list containing the squares of the numbers 1 to 5 (that is,
1, 4, 9, 16, 25), by starting from an empty list and adding one square at a time
in a loop. Then print each element of the finished list on its own line, in
order.

Print nothing else.

## Exact expected output

```
1
4
9
16
25
```
