# Task: shape areas

A shape is either a **circle** described by its radius, or a **rectangle**
described by its width and height. Model these two shapes as one type, and write
one `area` function that returns the area of any shape — using π ≈ 3 and integer
arithmetic, so a circle of radius `r` has area `3 * r * r`.

For a circle of radius 2 and a rectangle 3 by 5, print each area on its own line,
in that order.

Print nothing else.

## Exact expected output

```
12
15
```
